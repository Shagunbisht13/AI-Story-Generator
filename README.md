# AI Story Generator

## Overview

AI Story Generator is a Generative AI-based application that creates creative stories from user-provided prompts. The system uses Natural Language Processing (NLP) and Large Language Models (LLMs) to generate engaging and context-aware stories.

## Features

* AI-powered story generation
* Custom user prompts
* Multiple story themes and genres
* Simple and interactive interface

## Technologies Used

Python, Streamlit, Generative AI, NLP, OpenAI API, Git, GitHub

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
